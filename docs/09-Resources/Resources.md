## Resouces

ESP32 Python code ran in Thonny. Used to make the motor react to certain UART messages received from other teammates' subsystems.
Functionality includes:
- The ability for the motor to go full speed, slow, or stop using PWM when receiving a sensor value from JT's sensor value
- The ability to change directions based on an HMI button press message from Abriana's subsystem
- Sending the current motor status to Don's hall effect subsystem
- Able to pass along messages not meant for my ID designation and parse messages that do not belong to any of the team's IDs
- 
The Python code is available for download in a zip file [*here*](314finalcode.zip)
